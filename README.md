# Running the app

Clone repo
```sh
  git clone git@github.com:lucasgrocha/lucasgrocha.dev.git
```

cd into repo
```sh
  cd lucasgrocha.dev
```

Update the submodule git themes
```sh
  git submodule update --init
```

Run server
```sh
  hugo server -v
```

or with docker
```sh
  docker-compose up
```
